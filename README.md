# 📱 WhatsApp Reader - קורא הודעות וואטסאפ

קורא הודעות וואטסאפ בזמן אמת עם תמיכה מלאה בעברית (RTL), תמונות, סרטונים, הודעות קוליות וקישורים.

![Hebrew RTL Support](https://img.shields.io/badge/Hebrew-RTL%20Support-blue)
![Node.js](https://img.shields.io/badge/Node.js-14%2B-green)

## ✨ תכונות

- ✅ **עברית מימין לשמאל** - תצוגה נכונה בעברית בדף HTML
- ✅ **תמונות וסטיקרים** - הצגת preview עם אפשרות להגדלה
- ✅ **סרטונים** - ניגון אוטומטי בדף
- ✅ **הודעות קוליות** - נגן אודיו מובנה
- ✅ **קישורים קליקבילים** - כל קישור ניתן ללחיצה
- ✅ **שמירת סשן** - לא צריך לסרוק QR בכל פעם
- ✅ **ריענון אוטומטי** - הדף מתרענן כל 3 שניות

## 🚀 התקנה

### דרישות מקדימות
- [Node.js](https://nodejs.org/) גרסה 14 ומעלה
- חשבון וואטסאפ פעיל בטלפון

### שלבי התקנה

```bash
# 1. שכפל את הפרויקט
git clone <repository-url>
cd whatsapp-reader

# 2. התקן את החבילות
npm install
```

## 📖 שימוש

### הפעלה ראשונה (קישור מכשיר)

```bash
npm start
```

1. יופיע **קוד QR** בטרמינל
2. פתח את **וואטסאפ בטלפון**
3. לך ל: **הגדרות** → **מכשירים מקושרים** → **קשר מכשיר**
4. **סרוק את קוד ה-QR** עם הטלפון
5. ההודעות יתחילו להופיע!

### צפייה בהודעות

לאחר ההתחברות, פתח את הקובץ `messages.html` בדפדפן:
- ההודעות מוצגות בעברית מימין לשמאל
- תמונות וסרטונים מוצגים אוטומטית
- קישורים ניתנים ללחיצה
- הדף מתרענן אוטומטית

### הפעלות נוספות

לאחר הקישור הראשוני, הסשן נשמר אוטומטית.  
בהפעלות הבאות לא תצטרך לסרוק QR שוב:

```bash
npm start
```

### קריאת היסטוריית צ'אטים

```bash
npm run history
```

### איפוס סשן (אם נדרש קישור מחדש)

```bash
npm run reset
npm start
```

## 📁 מבנה הפרויקט

```
whatsapp-reader/
├── index.js              # הסקריפט הראשי
├── read-chat-history.js  # קריאת היסטוריה
├── package.json          # הגדרות הפרויקט
├── messages.html         # דף ההודעות (נוצר אוטומטית)
├── media/                # תמונות וסרטונים (נוצר אוטומטית)
└── .wwebjs_auth/         # נתוני הסשן (נוצר אוטומטית)
```

## ⚠️ הערות חשובות

1. **הטלפון חייב להיות מחובר לאינטרנט** - וואטסאפ Web דורש חיבור לטלפון
2. **אל תשתף את תיקיית `.wwebjs_auth`** - היא מכילה את נתוני הסשן שלך
3. **תיקיית `media`** - מכילה את כל התמונות והסרטונים שהתקבלו

## 🔧 פתרון בעיות

### קוד QR לא מופיע
- ודא שהטרמינל פתוח ונראה
- נסה להריץ `npm run reset` ואז `npm start`

### תמונות לא מוצגות
- ודא שאתה פותח את `messages.html` מאותה תיקייה
- בדוק שתיקיית `media` קיימת

### שגיאת התחברות
```bash
npm run reset
npm start
```

### עברית מופיעה הפוך
- פתח את `messages.html` בדפדפן (לא בטרמינל)
