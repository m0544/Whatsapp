# 📱 WhatsApp Reader - קורא הודעות וואטסאפ

קורא הודעות וואטסאפ בזמן אמת עם ממשק WEB מעוצב, תמיכה מלאה בעברית (RTL), תמונות, סרטונים, הודעות קוליות, קישורים עם תצוגה מקדימה ורשימת משימות.

![Hebrew RTL Support](https://img.shields.io/badge/Hebrew-RTL%20Support-blue)
![Node.js](https://img.shields.io/badge/Node.js-18%2B-green)

## ✨ תכונות

- ✅ **ממשק WEB מעוצב** - פתח בדפדפן בכתובת http://localhost:3000
- ✅ **עברית מימין לשמאל** - תצוגה נכונה בעברית
- ✅ **תמונות וסטיקרים** - הצגת preview עם אפשרות להגדלה
- ✅ **סרטונים** - ניגון אוטומטי בדף
- ✅ **הודעות קוליות** - נגן אודיו מובנה
- ✅ **קישורים עם Preview** - תצוגה מקדימה של קישורים
- ✅ **תגובות להודעות** - אפשרות להגיב ישירות מהממשק
- ✅ **רשימת משימות (TODO)** - סמן הודעות כמשימות לביצוע
- ✅ **שמירת סשן** - לא צריך לסרוק QR בכל פעם
- ✅ **עדכון אוטומטי** - הודעות חדשות מופיעות אוטומטית

## 🚀 התקנה מהירה (Windows)

### למשתמשים חדשים - הכי קל!

1. **הורד את הפרויקט** מ-GitHub (לחץ על Code → Download ZIP)
2. **חלץ את הקבצים** לתיקייה
3. **לחץ פעמיים על `setup.bat`**
4. הסקריפט יבדוק אם Node.js מותקן ויתקין את הכל אוטומטית!

> 💡 אם Node.js לא מותקן, ייפתח דף הורדה - התקן ואז הרץ שוב את setup.bat

---

## 📖 שימוש

### הפעלה ראשונה

1. הרץ את **`setup.bat`** (או `npm start` בטרמינל)
2. יופיע **קוד QR** בטרמינל
3. פתח את **וואטסאפ בטלפון**
4. לך ל: **הגדרות** → **מכשירים מקושרים** → **קשר מכשיר**
5. **סרוק את קוד ה-QR** עם הטלפון
6. פתח בדפדפן: **http://localhost:3000**

### הפעלות נוספות

לאחר הקישור הראשוני, הסשן נשמר אוטומטית.
פשוט הרץ את **`run.bat`** (או `npm start`).

## 🖥️ התקנה ידנית (לכל מערכות ההפעלה)

### דרישות מקדימות
- [Node.js](https://nodejs.org/) גרסה 18 ומעלה
- חשבון וואטסאפ פעיל בטלפון

### שלבי התקנה

```bash
# 1. שכפל את הפרויקט
git clone https://github.com/m0544/Whatsapp.git
cd Whatsapp

# 2. התקן את החבילות
npm install

# 3. הרץ
npm start
```

## 📁 מבנה הפרויקט

```
Whatsapp/
├── index.js              # הסקריפט הראשי
├── read-chat-history.js  # קריאת היסטוריה
├── package.json          # הגדרות הפרויקט
├── setup.bat             # סקריפט התקנה (Windows)
├── run.bat               # סקריפט הפעלה (Windows)
├── media/                # תמונות וסרטונים (נוצר אוטומטית)
├── todos.json            # רשימת משימות (נוצר אוטומטית)
└── .wwebjs_auth/         # נתוני הסשן (נוצר אוטומטית)
```

## ⚠️ הערות חשובות

1. **הטלפון חייב להיות מחובר לאינטרנט** - וואטסאפ Web דורש חיבור לטלפון
2. **אל תשתף את תיקיית `.wwebjs_auth`** - היא מכילה את נתוני הסשן שלך
3. **תיקיית `media`** - מכילה את כל התמונות והסרטונים שהתקבלו
